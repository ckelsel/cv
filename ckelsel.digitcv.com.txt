### 软件自动更新程序 
我在此项目负责升级程序的设计、开发和测试，实现不同的Windows软件升级，包括应用、服务、驱动。这个项目中，最困难的问题保证升级成功，我首先备份原始的安装文件，接着使用RSA签名来校验升级包，MD5校验升级包内的文件，替换目标文件，最后使用MD5校验目标文件。如果升级失败，则回滚。这个项目中，我最自豪的技术是XML，使用XML定义软件安装文件列表。

### Software automatic update program 
I am responsible for design and development the project to achieve different
Windows software upgrade, 
including applications, services, drivers.
The most difficult problem in this project is ensure a successful upgrade.
I first back up the original installation files, and then use the RSA signature to verify the upgrade package, MD5 check upgrade package files, replace the target file, and finally use the MD5 to verify the target file. 
If the upgrade fails, start rollback operation.
For this project, I am most proud of the technology is XML,
Use XML to define the software installation file list.
Do not use this technique only upgrade specific program. Using this technique you can upgrade different software, 
   The only condition is provide a list of software installation files.

## 福建升腾资讯有限公司（ 2012年6月 ~ 2013年12月 ）

### 基于ARM平台开发Xred远程桌面协议Linux客户端
我在此项目负责协议客户端从x86平台移植到ARM平台，使用XLIB实现二元、三元光栅操作，实现远程桌面协议客户端。这个项目中，最困难的问题保证桌面操作流畅，我采取了合并鼠标操作降低带宽，合并多次刷新，刷新频率30-40ms一帧，使用硬件加速绘制画面，达到用户操作流畅。这个项目中，我最自豪的是降低服务端带宽，优化服务端驱动cache机制和发送频率。驱动cache机制默认cache所有位图，但是全屏位图cache命中低，浪费带宽。另一方面，发送频率越快cache命中率越低，降低发送频率，提高cache命中率。优化后，数据量降低20-30%，从本质上降低带宽。

### Xred remote display protocol Linux client
I am responsible for adapt client from x86 platform to arm platform and
implement Raster Operations by using XLIB. 
The most difficult problem in this project is ensure smooth operation.
To make user happy, I merge the mouse position and multiple refresh area, also maintain a refresh rate of 30-40ms and use hardware acceleration.

For this project, the most proud of is reduce the bandwidth of server side,
by optimizing server drive cache mechanism and the transmission frequency.
Driver caches all bitmaps, while full-screen bitmaps is useless that cost a
lot of bandwidth. 
On the other hand, The faster the transmission frequency, the lower the cache
hit rate.Reduce the transmission frequency, improve cache hit rate. After
optimization, bandwidth is reduced by 20-30%.
